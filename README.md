# Instalação


~~~html
# Copie e cole no seu terminal 

$ git clone https://github.com/OttoSouza/ignite_movies_chapter_I.gitgit
~~~

~~~html
# Acesse a pasta
$ cd ignite_movies_chapter_I

# Instale as dependencias 
$ yarn install

# Execute
$ yarn start

# Acessa pelo  browser 
$ https://localhost:8080
~~~
